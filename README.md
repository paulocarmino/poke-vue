# Tecnologias envolvidas

- VueJS
- GraphQL (Apollo Client)
- Quasar

# Rodando o projeto

```
# Clone esse repositorio
git clone https://github.com/paulocarmino/poke-vue

# Entre no repositorio
cd poke-vue

# Instale as dependencias
yarn install

# Rode o app
yarn serve
```
